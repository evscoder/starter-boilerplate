* Node >= 12.0.0
* Gulp cli >= 2.3.0

## Guide
#### Install dependencies
```commandline
npm install
```

#### Start local and hot-reloads
```commandline
npm start
```
```commandline
Local server http://localhost:8080/
```

#### Create build
```commandline
npm run build
```
