# Starter Boilerplate
Modular builder for frontend development based on Gulp and Webpack including modern tools.

## Requirements
* Node >= 10.0.0
* Gulp cli >= 2.0.1

## Guide
#### Install dependencies
```commandline
npm install
```

#### Start local and hot-reloads
```commandline
npm start
```
```commandline
Local server http://localhost:8080/
```

#### Create build
```commandline
npm run build
```
